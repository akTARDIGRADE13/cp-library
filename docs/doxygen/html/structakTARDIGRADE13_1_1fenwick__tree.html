Detailed Description

template<class T>
struct akTARDIGRADE13::fenwick_tree< T >

Fenwick Tree（Binary Indexed Tree）

点更新および区間和クエリを高速に処理するためのデータ構造。

  • 外部から見えるインデックスは 0-indexed
  • 内部では 1-indexed の Fenwick Tree を使用
  • 半開区間 [l, r) を基本とする
Template Parameters
T加算・比較が可能な型。
Note
lower_bound / upper_bound を使用する場合、 木に格納される値は **非負**である必要がある。
